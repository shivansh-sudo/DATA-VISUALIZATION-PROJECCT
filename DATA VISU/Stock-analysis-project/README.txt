 Stock Market Data Analysis Project

 What I Did:
- Collected stock data using the yfinance library.
- Stored it in CSV format.
- Analyzed price trends and created a 3-day Moving Average.
- Visualized data using matplotlib.

 Tools Used:
- Python
- yfinance
- pandas
- matplotlib

 How to Run:
1. Run stock_scraper.py to collect stock data.
2. Run analyze_data.py to analyze and visualize trends.
